# Installation Guide

## Quick Install

```bash
pip install statclean
```

## Requirements

- Python 3.7+
- numpy >= 1.19.0
- pandas >= 1.2.0
- matplotlib >= 3.3.0
- seaborn >= 0.11.0
- scipy >= 1.6.0
- tqdm >= 4.60.0

## Development Install

```bash
git clone https://github.com/SubaashNair/StatClean.git
cd StatClean
pip install -e .
```

## Verification

```python
from statclean import StatClean
print("Installation successful!")
```
